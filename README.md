# 🚀 Gzip Tools（IntelliJ 插件）

一款用于 IntelliJ IDEA 的智能数据处理工具插件，支持 GZIP / Base64 / JSON / XML 一键识别和处理。

---

## 📌 功能介绍

### ✅ 智能处理（推荐 ⭐⭐⭐⭐⭐）

一键自动识别并处理：

- ✅ GZIP + Base64 → 自动解压
- ✅ Base64 → 自动解码
- ✅ JSON → 自动格式化
- ✅ XML → 自动格式化

👉 无需手动判断数据类型，一键即可完成处理。

---

### ✅ 手动功能

- ✔ Uncompress（GZIP + Base64 解压）
- ✔ Compress（带时间戳）
- ✔ Compress（无时间戳，稳定模式）

---

### ✅ UI 特性

- ✅ 简洁美观的弹窗界面
- ✅ 支持内容自动换行与滚动
- ✅ 一键复制结果
- ✅ Format / Restore 切换（格式化 ↔ 还原）
- ✅ 实时展示内容大小
- ✅ 按钮状态交互（高亮 / 重置）

---

## 🖱 使用方式

1. 在编辑器中选中一段文本
2. 右键点击，选择：Gzip Tools
